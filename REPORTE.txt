Reporte Técnico de Seguridad – Campaña de Phishing (Smishing) + Análisis de Infraestructura AWS
1. Resumen Ejecutivo

Se identificó una campaña activa de phishing mediante SMS (smishing) que utiliza técnicas de evasión basadas en redirecciones y hosting en infraestructura cloud (AWS S3 + CloudFront).

El análisis técnico permitió:

Confirmar la cadena completa de ataque
Identificar el uso de redirectores para ocultar el payload
Analizar la infraestructura backend (S3 + CDN)
Detectar controles de seguridad (WAF / rate limiting)
Enumerar rutas sensibles en el servidor objetivo
2. Alcance del análisis
-Activos analizados

URL acortada:

https://did.li/c5bsl

Dominio final:

https://www.xy321.club
3. Vector de ataque inicial (Smishing)
-Evidencia

Mensaje SMS:

“Su cuenta ha recibido 300.000 COP$. Por favor, revise su saldo lo antes posible. https://did.li/c5bsl”
-Análisis
Uso de ingeniería social basada en urgencia financiera
Inclusión de URL acortada para evasión
Mensaje diseñado para inducir interacción inmediata
4. Análisis de redirección (Redirect Chain)
-Prueba realizada
curl -I https://did.li/c5bsl
-Resultado
HTTP/2 301
location: https://www.xy321.club?tr=z34&r=18
server: AmazonS3
via: CloudFront
-Interpretación
Se confirma redirección HTTP 301 (permanente)
Uso de:
Amazon S3 como backend
CloudFront como CDN
⚠️ Hallazgos
Uso de redirector para ocultar destino final

Parámetros detectados:

?tr=z34&r=18

Posible:

Tracking de campaña
Identificador de víctima
5. Análisis de infraestructura del dominio final
5.1 Resolución HTTP vs HTTPS
Pruebas:
curl -I http://www.xy321.club
curl -I https://www.xy321.club
-Resultados
HTTP:
301 Moved Permanently → https://www.xy321.club/
HTTPS:
HTTP/2 200 OK
server: AmazonS3
x-cache: Hit from cloudfront
-Interpretación
Redirección HTTP → HTTPS correctamente implementada
Contenido servido desde:
Amazon S3 (hosting estático)
CloudFront (capa CDN)
5.2 Características del backend
Componente	Tecnología
Hosting	AWS S3
CDN	CloudFront
Tipo de app	Estática (HTML + JS)
5.3 Cache
x-cache: Hit from cloudfront
age: 15631

-Indica:

Contenido cacheado (~4 horas)
Posible vector teórico: cache poisoning
